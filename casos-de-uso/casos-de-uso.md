## Casos de Uso do Sistema Tanq

A lista a seguir detalha os casos de uso identificados para o sistema Tanq. Cada item representa uma interação específica que um ator (como o Motorista, Proprietário de Posto ou Administrador) pode realizar para atingir um objetivo.

* **UC01:** Realizar Logout
* **UC02:** Realizar Login
* **UC03:** Recuperar Senha
* **UC04:** Criar Conta de Motorista
* **UC05:** Criar Conta de Dono de Posto
* **UC06:** Consultar Ranking de Preços
* **UC07:** Acompanhar Avaliações
* **UC08:** Consultar Detalhes e Avaliações de um Posto
* **UC09:** Visualizar Postos no Mapa
* **UC10:** Resgatar Cupom de Desconto
* **UC11:** Avaliar Posto de Combustível
* **UC12:** Gerenciar Contas de Parceiros
* **UC13:** Moderar Conteúdo
* **UC14:** Gerenciar Cupons e Promoções
* **UC15:** Obter Recomendação via Chatbot
* **UC16:** Atualizar Preço com Foto
